# BankingProgram
# Introduction

A console-based banking application built with Java that allows users to manage their account balance through deposits, withdrawals and balance checks.

# Features

- Check account balance
- Deposit funds with validation
- Withdraw funds with insufficient funds check
- Handles negative number inputs
- Clean exit with goodbye message

# How To Run

- Clone this repository
- Open in IntelliJ IDEA
- Run Main.java

# Examples

- Start program => Banking menu appears
- Choose 1 => Shows current balance
- Choose 2 => Deposit amount
- Choose 3 => Withdraw amount
- Choose 4 => Exit program
